# Exterior Placards

**Print on weatherproof adhesive labels or laminate and attach to enclosure exterior.**

---

## Placard 1: Humidity Warning (Both Sites)

```
┌──────────────────────────────────────────────┐
│                                              │
│      ⚠  DO NOT OPEN THIS ENCLOSURE          │
│         WHEN HUMIDITY IS ABOVE 70%           │
│                                              │
│   Opening in high humidity allows moisture   │
│   to condense on electronics inside.         │
│                                              │
│   Check conditions before opening.           │
│   If you must open in humid conditions,      │
│   minimize time open and close immediately.  │
│                                              │
└──────────────────────────────────────────────┘
```

## Printing Notes

- **Material:** Weatherproof adhesive vinyl labels, or laminated cardstock
  with outdoor adhesive backing
- **Size:** Each placard approximately 10cm x 7cm (4" x 3")
- **Placement:**
  - Humidity warning: on enclosure door, eye level
  - Voltage warning (Jakarta): on enclosure door, above humidity warning
  - Site ID: on enclosure side, visible from ground
  - Do not power off: near the power input connection point
- **Both languages** on each placard (English on top, Bahasa below), or
  print separate placards for each language depending on space
- Replace if faded or damaged — UV degrades labels over time
