# Archive — Spring 2026 Indonesia Deployment Pre-Trip Planning

These docs were the pre-trip plan as of departure (April 12, 2026). They are
retained as historical reference. **For current status see the parent
[README.md](../README.md), [TODO.md](../TODO.md), and
[LESSONS_LEARNED.md](../LESSONS_LEARNED.md).**

What actually happened during the trip differed from this plan in two major
ways:

- **Jakarta was not deployed.** Permission for the intended site fell
  through during the trip. The station is built and software-ready but is
  back in the US pending IPB consultation on alternate site selection.
- **Sukabumi RTK survey failed twice.** Both the initial and re-survey
  attempts produced data with ~99 cm horizontal / ~139 cm vertical
  check-point noise, well above the 3 cm RTK gate. The site is operating
  on a salvage calibration pending an IPB total station re-survey.
